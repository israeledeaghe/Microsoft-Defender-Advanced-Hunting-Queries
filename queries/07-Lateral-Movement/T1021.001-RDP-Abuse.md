# Remote Desktop Protocol Abuse

**MITRE Technique:** T1021.001 - Remote Services: Remote Desktop Protocol  
**MITRE Tactic:** TA0008 - Lateral Movement  
**Severity:** High  
**False Positive Rate:** Medium  

## Description
Detects suspicious RDP connections on port 3389. Attackers use RDP extensively for lateral movement after obtaining valid credentials. Unexpected RDP connections — especially from non-standard source IPs or outside business hours — are strong indicators of lateral movement activity.

## KQL Query
```kql
DeviceNetworkEvents
| where Timestamp > ago(24h)
| where RemotePort == 3389 or LocalPort == 3389
| where ActionType == "ConnectionSuccess"
| project
    Timestamp,
    DeviceName,
    RemoteIP,
    RemotePort,
    LocalPort,
    InitiatingProcessFileName,
    InitiatingProcessAccountName
| order by Timestamp desc
```

## Results in Lab
- **Hit found:** No results
- **Verdict:** Expected baseline — no RDP connections observed
- **Notes:** Query validated against live MDE P2 telemetry on Windows 11 Pro

## Tuning Notes
- Whitelist known jump servers and IT admin source IPs
- Combine with T1078 logon events query for complete lateral movement picture
- Flag RDP connections outside business hours as highest priority
