# Suspicious Outbound Network Connections

**MITRE Technique:** T1071.001 - Application Layer Protocol: Web Protocols  
**MITRE Tactic:** TA0011 - Command and Control  
**Severity:** High  
**False Positive Rate:** Low  

## Description
Detects processes making outbound connections on ports commonly used by attackers for command and control frameworks such as Metasploit, Cobalt Strike, and custom reverse shells. Port 4444 is the default Metasploit listener port. Port 31337 is historically associated with elite hacker tools.

## KQL Query
```kql
DeviceNetworkEvents
| where Timestamp > ago(24h)
| where ActionType == "ConnectionSuccess"
| where RemotePort in (4444, 5555, 8080, 8443, 1234, 31337, 9001, 9002)
| where InitiatingProcessFileName !in~ (
    "chrome.exe",
    "msedge.exe",
    "firefox.exe",
    "brave.exe"
)
| project
    Timestamp,
    DeviceName,
    InitiatingProcessFileName,
    InitiatingProcessCommandLine,
    RemoteIP,
    RemotePort,
    RemoteUrl,
    LocalIP,
    LocalPort
| order by Timestamp desc
```

## Results in Lab
- **Hit found:** No results
- **Verdict:** Expected baseline — no connections to suspicious ports observed
- **Notes:** Query validated against live MDE P2 telemetry on Windows 11 Pro

## Tuning Notes
- Ports 8080 and 8443 may generate false positives in environments with web proxies — add proxy IPs to exclusion list
- Correlate RemoteIP with threat intelligence feeds (VirusTotal, AbuseIPDB) for enrichment
- Expand port list based on known C2 frameworks in your threat model
