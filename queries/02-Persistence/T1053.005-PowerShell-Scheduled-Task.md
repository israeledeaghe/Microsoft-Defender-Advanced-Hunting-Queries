# Scheduled Task Creation via PowerShell

**MITRE Technique:** T1053.005 - Scheduled Task/Job  
**MITRE Tactic:** TA0003 - Persistence  
**Severity:** High  
**False Positive Rate:** Low  

## Description
Detects scheduled tasks created silently via PowerShell cmdlets. While schtasks.exe is the traditional tool for task creation, attackers increasingly use PowerShell's ScheduledTasks module to create tasks programmatically, bypassing command-line based detections.

## KQL Query
```kql
DeviceProcessEvents
| where Timestamp > ago(24h)
| where FileName in~ ("powershell.exe", "pwsh.exe")
| where ProcessCommandLine has_any (
    "New-ScheduledTask",
    "Register-ScheduledTask",
    "ScheduledTasks",
    "New-ScheduledTaskAction",
    "New-ScheduledTaskTrigger"
)
| project
    Timestamp,
    DeviceName,
    AccountName,
    FileName,
    ProcessCommandLine,
    InitiatingProcessFileName
| order by Timestamp desc
```

## Results in Lab
- **Hit found:** No results
- **Verdict:** Expected baseline — no PowerShell scheduled task creation observed
- **Notes:** Query validated against live MDE P2 telemetry on Windows 11 Pro

## Tuning Notes
- Whitelist known automation scripts by AccountName
- Correlate with T1053.005 schtasks.exe query for full scheduled task coverage
- Flag tasks scheduled to run from temp or AppData directories as highest priority
