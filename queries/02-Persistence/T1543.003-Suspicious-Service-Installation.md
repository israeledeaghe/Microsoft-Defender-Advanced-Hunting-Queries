# Suspicious Service Installation

**MITRE Technique:** T1543.003 - Create or Modify System Process: Windows Service  
**MITRE Tactic:** TA0003 - Persistence  
**Severity:** High  
**False Positive Rate:** Medium  

## Description
Detects new Windows services being created or modified via sc.exe. Attackers install malicious services to maintain persistence across reboots and execute payloads with SYSTEM privileges. Legitimate software installers also create services, so context is critical when investigating hits.

## KQL Query
```kql
DeviceProcessEvents
| where Timestamp > ago(24h)
| where FileName =~ "sc.exe"
| where ProcessCommandLine has_any (
    "create",
    "config",
    "failure"
)
| where ProcessCommandLine !has "query"
| project
    Timestamp,
    DeviceName,
    FileName,
    ProcessCommandLine,
    InitiatingProcessFileName,
    InitiatingProcessAccountName
| order by Timestamp desc
```

## Results in Lab
- **Hit found:** No results
- **Verdict:** Expected baseline — no suspicious service installations observed
- **Notes:** Query validated against live MDE P2 telemetry on Windows 11 Pro

## Tuning Notes
- Exclude known software installers by whitelisting InitiatingProcessFileName
- Flag services pointing to temp directories or user-writable paths as highest priority
- Correlate with DeviceRegistryEvents to confirm service registry key creation
