# Suspicious Scheduled Task Creation

**MITRE Technique:** T1053.005 - Scheduled Task/Job  
**MITRE Tactic:** TA0003 - Persistence  
**Severity:** High  
**False Positive Rate:** Medium  

## Description
Detects scheduled tasks created via schtasks.exe. Attackers use scheduled tasks to maintain persistence across reboots, execute payloads at specific times, or run malicious commands under different user contexts. This is one of the most commonly abused persistence mechanisms on Windows.

## KQL Query
```kql
DeviceProcessEvents
| where Timestamp > ago(24h)
| where FileName =~ "schtasks.exe"
| where ProcessCommandLine has_any ("/create", "-create")
| extend 
    TaskName = extract(@"/tn\s+([^\s/]+)", 1, ProcessCommandLine),
    TaskCommand = extract(@"/tr\s+([^\s/]+)", 1, ProcessCommandLine),
    TaskTrigger = extract(@"/sc\s+(\w+)", 1, ProcessCommandLine)
| project
    Timestamp,
    DeviceName,
    AccountName,
    ProcessCommandLine,
    TaskName,
    TaskCommand,
    TaskTrigger,
    InitiatingProcessFileName,
    InitiatingProcessAccountName
| order by Timestamp desc
```

## Results in Lab
- **Hit found:** No results
- **Verdict:** Expected baseline — no scheduled task creation observed in last 24 hours
- **Notes:** Query validated against live MDE P2 telemetry on Windows 11 Pro

## Tuning Notes
- Exclude known software installers (e.g. Windows Update, antivirus) by whitelisting InitiatingProcessFileName
- Flag tasks with commands pointing to temp directories, AppData, or unusual paths
- Correlate with DeviceFileEvents to check if the task executable was recently dropped
