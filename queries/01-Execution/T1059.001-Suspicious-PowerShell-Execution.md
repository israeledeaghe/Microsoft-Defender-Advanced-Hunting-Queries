# Suspicious PowerShell Execution

**MITRE Technique:** T1059.001 - Command and Scripting Interpreter: PowerShell  
**MITRE Tactic:** TA0002 - Execution  
**Severity:** High  
**False Positive Rate:** Medium  

## Description
Detects PowerShell launched with flags commonly used by attackers to evade detection, download payloads, or execute encoded commands. Flags include encoded commands, hidden windows, and download cradles.

## KQL Query
```kql
DeviceProcessEvents
| where Timestamp > ago(24h)
| where FileName =~ "powershell.exe" or FileName =~ "pwsh.exe"
| where ProcessCommandLine has_any (
    "-EncodedCommand",
    "-enc ",
    "-NoProfile",
    "-NonInteractive",
    "-WindowStyle Hidden",
    "IEX",
    "Invoke-Expression",
    "DownloadString",
    "FromBase64String"
)
| project 
    Timestamp,
    DeviceName,
    AccountName,
    FileName,
    ProcessCommandLine,
    InitiatingProcessFileName,
    InitiatingProcessCommandLine
| order by Timestamp desc
```

## Results in Lab
- **Hit found:** Yes — 1 result
- **Process:** powershell.exe launched by senseir.exe (MDE sensor) as SYSTEM
- **Verdict:** True positive with legitimate context (MDE onboarding activity)

## Tuning Notes
- Exclude `senseir.exe` and `MsSense.exe` as initiating processes to reduce MDE sensor noise
- Whitelist known software deployment tools in your environment
- Focus on user-context executions (AccountName != "system") for higher fidelity alerts

## Tuned Query (Production-Ready)
```kql
DeviceProcessEvents
| where Timestamp > ago(24h)
| where FileName =~ "powershell.exe" or FileName =~ "pwsh.exe"
| where ProcessCommandLine has_any (
    "-EncodedCommand",
    "-enc ",
    "-WindowStyle Hidden",
    "IEX",
    "Invoke-Expression",
    "DownloadString",
    "FromBase64String"
)
| where InitiatingProcessFileName !in~ ("senseir.exe", "MsSense.exe", "SenseCncProxy.exe")
| where AccountName != "system"
| project 
    Timestamp,
    DeviceName,
    AccountName,
    FileName,
    ProcessCommandLine,
    InitiatingProcessFileName,
    InitiatingProcessCommandLine
| order by Timestamp desc
```
