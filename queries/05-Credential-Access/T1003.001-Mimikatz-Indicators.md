# Mimikatz Credential Dumping Indicators

**MITRE Technique:** T1003.001 - OS Credential Dumping  
**MITRE Tactic:** TA0006 - Credential Access  
**Severity:** Critical  
**False Positive Rate:** Low  

## Description
Detects Mimikatz-specific command strings and known module names in process command lines. Mimikatz is the most widely used credential dumping tool, capable of extracting plaintext passwords, NTLM hashes, Kerberos tickets, and more from Windows memory. Any hit on this query should be treated as a critical incident.

## KQL Query
```kql
DeviceProcessEvents
| where Timestamp > ago(24h)
| where ProcessCommandLine has_any (
    "sekurlsa",
    "logonpasswords",
    "lsadump",
    "dcsync",
    "kerberos::list",
    "kerberos::ptt",
    "privilege::debug",
    "token::elevate",
    "crypto::certificates",
    "vault::cred"
)
| project
    Timestamp,
    DeviceName,
    FileName,
    ProcessCommandLine,
    InitiatingProcessFileName,
    InitiatingProcessAccountName
| order by Timestamp desc
```

## Results in Lab
- **Hit found:** No results
- **Verdict:** Expected baseline — no Mimikatz indicators observed
- **Notes:** Query validated against live MDE P2 telemetry on Windows 11 Pro

## Tuning Notes
- This query has an extremely low false positive rate — any hit is critical
- Extend the keyword list with obfuscated variants of Mimikatz commands
- Correlate with DeviceLogonEvents to check for subsequent authentication activity
- Cross-reference with network events to detect potential pass-the-hash lateral movement
